## Introduction

**Handmade & Precious** is an ecommerce app made with Flutter using server function and realm database.

---

## :sparkles: Main Features

- Authentication.
- Add items to your whishlist
- Search Products
- Infinite list (Pagination)
- Sort with category
- Profile 
- Order
- Demo Payment Gateway
